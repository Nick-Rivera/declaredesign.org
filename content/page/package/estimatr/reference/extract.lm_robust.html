Extract model data for texreg package

Prepares an "lm_robust" object for the texreg package. This is largely a clone of the extract.lm method.

extract.lm_robust(model, include.ci = TRUE, include.rsquared = TRUE,
  include.adjrs = TRUE, include.nobs = TRUE, include.fstatistic = FALSE,
  include.rmse = TRUE, ...)

Arguments

model

an object of class lm_robust
include.ci

logical. Defaults to TRUE
include.rsquared

logical. Defaults to TRUE
include.adjrs

logical. Defaults to TRUE
include.nobs

logical. Defaults to TRUE
include.fstatistic

logical. Defaults to TRUE
include.rmse

logical. Defaults to TRUE
...

unused
